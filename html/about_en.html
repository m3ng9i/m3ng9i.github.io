About me

Hi, my name is Meng Qi (孟琦). I'm Chinese and live in China.

I'm not a professional developer but I enjoy coding and making things. Currently my hobby is the Go programming language and I published some projects on Github:

  • Ran: a simple static web server written in Go
  • Feedreader: a Go package for parsing RSS 2.0 and Atom 1.0 feed
  • Godi: a tool for gathering a go package's dependency information
  • IP-resolver: a tool for getting a domain's IPs from multiple name servers. (Also my first Github project)
  • QReader: an RSS reader written in Go and JavaScrip (the UI and repository description are in Chinese)
